# ❓ FAQ - Часто задаваемые вопросы

## 🚀 Запуск и установка

### Q: При запуске пишет "ModuleNotFoundError"
**A:** Зависимости не установлены. Выполни:
```bash
pip install -r requirements.txt
```

### Q: Как запустить бота на Windows?
**A:** 
```cmd
set BOT_TOKEN=123456:ABCDefGHIJK
set FACEIT_API_KEY=faa2d51c-d11e-431f-a20a-ddec6f8d7f41
python faceit_analyzer_bot.py
```

Или используй PowerShell:
```powershell
$env:BOT_TOKEN="123456:ABCDefGHIJK"
$env:FACEIT_API_KEY="faa2d51c-d11e-431f-a20a-ddec6f8d7f41"
python faceit_analyzer_bot.py
```

### Q: Какая версия Python нужна?
**A:** Python 3.10 или выше. Проверь версию:
```bash
python3 --version
```

## 🔑 Токены и ключи

### Q: Где получить Telegram Bot Token?
**A:**
1. Открой Telegram → найди @BotFather
2. Напиши `/newbot`
3. Дай имя боту
4. Дай username (уникальный, заканчивается на _bot)
5. Получишь токен вида: `123456:ABCDefGHIJKLMNOpqrstuVWXYZ`

### Q: Где получить Faceit API Key?
**A:**
1. Зайди на https://www.faceit.com/en/settings/account
2. Проскролл вниз до "API" или "Developer"
3. Нажми "Generate API Key"
4. Скопируй ключ

### Q: Что делать если потерял токены?
**A:**
- **Bot Token:** Зайди в @BotFather, `/mybots` → выбери бота → Revoke current token → Generate new token
- **Faceit API Key:** На странице https://www.faceit.com/en/settings/account нажми "Regenerate API Key"

### Q: Токен виден в истории Telegram/GitHub. Что делать?
**A:** СРОЧНО:
1. Зайди в @BotFather
2. Выбери бота → Revoke current token → Generate new token
3. Для Faceit: https://www.faceit.com/en/settings/account → Regenerate API Key
4. Обнови токены в коде/переменных

## 🔴 Ошибки при анализе

### Q: "Invalid API Key" или "Unauthorized"
**A:** 
- Проверь, что ключ скопирован полностью (без пробелов)
- Попытайся сгенерировать новый ключ на Faceit
- Убедись, что используешь правильный формат переменных окружения

### Q: "Player not found"
**A:**
- Проверь написание ника (может быть чувствительно к регистру в некоторых случаях)
- На Faceit может быть несколько игроков с похожим ником
- Попробуй вставить полную ссылку на профиль: `https://www.faceit.com/en/players/NICKNAME`

### Q: "Connection timeout" или "Connection refused"
**A:**
- Проверь интернет соединение
- Faceit API может быть недоступен (редко)
- Попробуй через VPN если у тебя блокировка

### Q: Статистика показывается неправильно
**A:**
- Faceit API может кэшировать данные (обновляется раз в 5-10 мин)
- Попробуй анализировать через пару минут
- Убедись, что ник игрока указан правильно

## 💾 База данных

### Q: Где сохраняются анализы?
**A:** В файле `faceit_stats.db` (SQLite база). Находится в той же папке что и скрипт.

### Q: Как удалить историю анализов?
**A:**
```bash
rm faceit_stats.db
```
Новая база создастся при следующем анализе.

### Q: Как экспортировать статистику?
**A:** Используй скрипт из EXAMPLES.md:
```python
export_stats('skywhyelo')  # Создаст skywhyelo_stats.csv
```

### Q: База повреждена, что делать?
**A:**
```bash
rm faceit_stats.db  # Удали старую
python faceit_analyzer_bot.py  # Создаст новую
```

## 📊 Функциональность

### Q: Почему HS% не совпадает с профилем?
**A:**
- Faceit может считать HS% по-разному для разных периодов (lifetime/last month)
- Бот показывает lifetime (всё время)
- Обычно совпадает в течение часа

### Q: Можно ли анализировать более 2 игроков одновременно?
**A:** Текущая версия анализирует 2 игроков. Но можешь:
- Сделать несколько анализов подряд
- Добавить функцию team_rating (см. EXAMPLES.md)

### Q: Что означают символы в сравнении карт?
**A:**
- 👤 = первый игрок лучше на этой карте
- 🤖 = второй игрок лучше
- ⚖️ = одинаково
- 📍 = один из игроков не играет на карте

### Q: Почему рекомендации могут быть неточными?
**A:**
- Базируются на lifetime статистике
- Не учитывают текущую форму
- Это общие советы, а не персональный коучинг

## 🤖 Бот не работает

### Q: Бот не отвечает на сообщения
**A:**
1. Проверь, что бот запущен (окно консоли должно работать)
2. Убедись что токен правильный
3. Перезагрузи бота: Ctrl+C → запусти заново
4. Проверь логи консоли на ошибки

### Q: Кнопки в меню не работают
**A:**
- Попробуй отправить команду вручную: `/start`
- Обнови Telegram (иногда есть баги с кнопками)
- Перезагрузи бота

### Q: "Timed out" при анализе
**A:**
- Faceit API медленный, может брать 10-30 сек
- Подожди (обычно ждет)
- Проверь интернет

### Q: Бот виснет при запуске
**A:**
- Может загружаться долго первый раз (зависимости)
- Если виснет > 1 мин - нажми Ctrl+C и перезагрузи
- Проверь интернет соединение

## 🔐 Безопасность

### Q: Безопасно ли делиться своим ником?
**A:** Да, ник (nickname) - публичная информация. Все анализируют по никам. Не делись:
- Bot Token (123456:ABC...)
- Faceit API Key (faa2d51c-...)
- Паролями от Telegram/Faceit

### Q: Может ли кто-то взломать бот используя API Key?
**A:** Теоретически да. Поэтому:
- Не публикуй ключи
- Используй переменные окружения (не hardcode)
- Регулярно регенерируй ключи
- Ограничивай доступ бота только своей аккаунту

### Q: Бот сохраняет мои данные?
**A:** 
- Только результаты анализов (ники, статистика)
- БД хранится локально (не в облаке)
- Никакие приватные данные не отправляются
